Installation Guide

1. Download

Get the ERDSetup.exe installation setup.

2. Run Setup

Open the installation file and press Next on the welcome screen.

3. Configure

Specify the installation directory and choose Start Menu shortcuts.

4. Finish

Complete the installation. You are ready to start modeling!

System Requirements

  • Operating System
    Microsoft Windows 64-bit
  • Memory
    2 GB RAM or higher
  • Storage
    64 GB HDD free space
  • Drivers
    ODBC Drivers for SQL Server, MySQL, or Snowflake (for database connectivity)
