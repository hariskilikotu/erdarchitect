Powerful Features

Everything you need for enterprise-grade data modeling.

Advanced Data Modeling

visualize and understand information structure effectively.

ER Modeling

Supports Entity Relationship Modeling to describe business requirements. Create illustrative sketches, workflows, and use case diagrams.

IE Notation

Full support for Information Engineering (IE) notation. Represent entities, attributes, and relationships with precise cardinality symbols.

Entities & Attributes

Define significant business objects (Entities) and their characteristics (Attributes). Easily manage structural and behavioral characteristics.

Relationships

Model complex associations: Identifying/Non-Identifying, Many-to-Many (with auto-resolution), Recursive, and Subtype relationships.

Keys Management

Comprehensive support for Primary, Unique, Foreign, Alternate, Composite, and Surrogate keys to ensure data integrity.

Database Engineering

Seamlessly bridge the gap between design and implementation.

Forward Engineering

Generate database schemas from your model. Use Instant DDL for quick scripts or the DDL Workbench for controlled execution.

Reverse Engineering

Import schemas from existing databases. Create diagrams from imported objects and perform extensive analysis.

Schema Comparison

Compare any combination of databases or ERD models. Visualize differences (Equal, Modified, Only on Left/Right) and generate update scripts.

Broad Connectivity

Connect to major database platforms.

SQL Server

Connect via ODBC Driver. Full support for forward/reverse engineering and schema comparison.

MySQL

Support for MySQL 8.0+ via ODBC Connector. Import tables, views, procedures, and functions.

Snowflake

Modern cloud data support. Connect using Snowflake ODBC driver with full warehouse and schema configuration.

Documentation & Reporting

Share knowledge with your team.

HTML Reports

Publish interactive HTML reports including diagrams for documentation and peer reviews.

CSV Export

Export model metadata to CSV format for further analysis and integration.

Diagram Export

Save diagrams as high-quality images (PNG, JPEG, BMP, GIF, TIFF) for external use.
